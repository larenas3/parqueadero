PROJECT TITLE:Control parqueadero
PURPOSE OF PROJECT: Permitir calcular el costo del servicio y llevar
    la cuenta del dinero recaudado
VERSION or DATE: 11-08-2017
AUTHORS: Lida Heliana Arenas Mogollón
USER INSTRUCTIONS: El usuario tiene un menú donde debe seleccionar
1) Calcular valor: Donde el usuario ingresa la hora de entrada, la hora de
    salida y el tipo de vehículo. El formato que se emplea para ingresar
    la hora es hora:minutos, ejemplo: 16:37
2) Cerrar: para salir del programa.
